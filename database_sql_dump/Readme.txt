Read me
* Create Database with name FIFA-19 into the postgreSQL server
* Using the Open File feature in the query tool, open the given create.sql file
* Run the file, all the tables will be created.
* On the table fifa_details, we will insert the provided data.csv file.
* Right click on the fifa_details, Click on the import/export option.
* Under the options tab, Slide the import option, provide the path for the csv file, slide the button for the header. Click on OK.
* Now the data should be inserted into the generated tables.
* Again, Using the Open File feature in the query tool, open the given load.sql file
* Run the file, all the data will be loaded.
* We are ready for the queries.
